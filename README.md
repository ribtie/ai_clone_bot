# ai_clone_bot
AI Clone Bot - цифровые двойники
ДЛЯ РАБОТЫ БОТА НУЖНЫ КУКИ ВАШЕГО БРАУЗЕРА:
1. Заходим на сайт vocloner.com
2. Логгинимся на сайт
3. Нажимаем F12, заходим в DevTools/Application/Cookies/PHPSESSID
4. В файле bot.py меняем в массиве cookies строчку PHPSESSID на ID вашей сессии
ДЛЯ РАБОТЫ ГОЛОСОВЫХ СООБЩЕНИЙ НЕОБХОДИМО СОЗДАТЬ КЛОН ГОЛОСА НА САЙТЕ vocloner.com;
Для Артема: вставляем ID голоса из ссылки сайта вида "https://vocloner.com/tts.php?voiceid=ID_ГОЛОСА" в поле "artem" файла bot.py; Для Максима - проводим те же операции в поле "max";
